# java-cli-maven-failsafe-spring-findbugs-pmd-jbehave-mockito-testng-car-data

## Description
A POC for maven app using testng
and jbehave and mockio framework
 with findbugs,
pmd and failsafe plugin.

## Tech stack
- java
- maven
	- spring
	- mockito
  - testng
  - findbugs
  - pmd
  - failsafe
  - jbehave

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
